# Notes

Aplicación de Notas simple desarrollada en Jetpack Compose, MVVM, Clean Arquitecture y Dagger Hilt,
con persistencia de datos en Room

## Características

* Operaciones básicas CRUD \(Create, Read, Update, Delete\)
